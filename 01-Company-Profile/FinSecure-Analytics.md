# FinSecure Analytics Ltd - Company Profile

## Overview
FinSecure Analytics Ltd is a UK-based financial technology company providing financial data analytics services to SMEs.

## Business Model
The company collects, processes, and analyzes financial data from clients through a cloud-based platform.

## Infrastructure
- Cloud environment (AWS/Azure)
- Web-based application
- Internal database systems
- Microsoft 365 (email and collaboration)
- Security monitoring via SIEM (Splunk)

## Key Assets
- Customer financial data
- Personally Identifiable Information (PII)
- Internal company data
- Application source code

## Security Objectives
- Protect confidentiality of financial data
- Ensure integrity of analytics results
- Maintain system availability
- Comply with GDPR and UK data protection laws

  ## Business Processes

- Data collection from SME clients
- Financial data processing and analysis
- Report generation via web platform
- User account management and authentication

## Users

- Internal Staff (Analysts, Developers, Admin)
- External Clients (SMEs)

## Locations

- Headquarters: United Kingdom
- Cloud Infrastructure: AWS/Azure (UK region)
