# ISO 27001 ISMS Implementation Project

## Overview
This project demonstrates a practical implementation of an Information Security Management System (ISMS) aligned with ISO/IEC 27001:2022.

The goal is to simulate a real-world organisation and develop key ISMS components including scope definition, risk assessment, policies, and control mapping.

---

## Organisation Scenario
**Company Name:** GenStudio Ltd  
**Industry:** Digital Creative & E-commerce Brand  
**Services:**
- AI-generated fashion content
- Social media automation (n8n workflows)
- E-commerce operations

---

## Objectives
- Implement a structured ISMS based on ISO 27001
- Perform risk assessment and treatment
- Develop core security policies
- Map controls to Annex A
- Demonstrate practical GRC capability

---

## ISMS Components

| Component | Description |
|----------|------------|
| ISMS Scope | Defines boundaries of the ISMS |
| Interested Parties | Identifies stakeholders and requirements |
| Security Policy | High-level information security direction |
| Risk Assessment | Identifies and evaluates risks |
| Risk Register | Tracks identified risks and treatments |
| Annex A Mapping | Maps controls to ISO 27001 |
